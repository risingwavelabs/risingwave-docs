## Query interfaces

You can query data from RisingWave using SELECT statements through multiple interfaces:

### Command line (psql)
Connect via `psql` and run your SELECT queries directly.

### RisingWave Console
Use the [RisingWave Console](/risingwave-console/use-sql-console) for a web-based SQL interface with syntax highlighting, query history, and result visualization.

### Other tools
Connect from various visualization tools and BI platforms. See [Query from visualization tools](/serve/query-from-visualization-tools) for more options.

## SELECT statement reference
For detailed SELECT statement syntax, see [SQL SELECT command](/sql/commands/sql-select).
