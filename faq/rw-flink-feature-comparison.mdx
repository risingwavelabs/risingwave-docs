This document provides a detailed feature-by-feature comparison between RisingWave 2.0 and Apache Flink 1.20. While both systems excel at stream processing, they have different architectural approaches and feature sets. This comparison helps users understand the similarities and differences between these two systems across various aspects including:

- Fundamental streaming concepts
- Data types and type system
- SQL query capabilities
- Window operations
- Joins and set operations
- DDL and DML statements
- Built-in functions
- System management features

<Note>
Features marked as "RisingWave-specific" indicate capabilities that are unique to RisingWave's architecture as a streaming database, while Flink-specific features reflect its nature as a stream processing framework.
</Note>
