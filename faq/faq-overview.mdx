## Still need help?

Can't find the answer you're looking for? Here's how to get support:

<Tabs>
  <Tab title="Quick answers">
    **Try these self-service options first:**
    - Ask in [#ask-ai channel](https://risingwave-community.slack.com/archives/C06R8DA3DGT) for AI-powered answers based on docs and community knowledge
    - Search the [Troubleshooting guide](/troubleshoot/overview) for common issues
    - Browse [GitHub discussions](https://github.com/risingwavelabs/risingwave/discussions) for community Q&A
  </Tab>

  <Tab title="Community & support">
    **Get help from the community:**
    - Join [Slack Community](https://www.risingwave.com/slack) for real-time help
    - Search or create [GitHub issues](https://github.com/risingwavelabs/risingwave/issues) to report bugs

    **RisingWave Cloud users:**
    - Check [Cloud FAQ](/cloud/faq) for Cloud-specific questions
    - Email [cloud-support@risingwave-labs.com](mailto:cloud-support@risingwave-labs.com) for incidents or urgent issues

    **Enterprise support:**
    - Contact [sales@risingwave-labs.com](mailto:sales@risingwave-labs.com) for premium support packages
  </Tab>
</Tabs>
