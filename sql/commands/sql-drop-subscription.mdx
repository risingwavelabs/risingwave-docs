---
title: "DROP SUBSCRIPTION"
description: "Use the `DROP SUBSCRIPTION` command to remove a subscription."
---

## Syntax

```sql
DROP SUBSCRIPTION [ IF EXISTS ] subscription_name;
```

## Parameters

| Parameter          | Description                                                                                                     |
| :----------------- | :-------------------------------------------------------------------------------------------------------------- |
| **IF EXISTS**      | Do not return an error if the subscription does not exist.                                                      |
| _subscription_name_ | The name of the subscription to be dropped.                                                                    |

## Examples

### Drop a subscription

```sql
DROP SUBSCRIPTION user_changes;
```

### Drop a subscription if it exists

```sql
DROP SUBSCRIPTION IF EXISTS user_changes;
```
