---
title: "DROP SUBSCRIPTION"
description: "Use the `DROP SUBSCRIPTION` command to remove a subscription."
---

## Syntax

```sql
DROP SUBSCRIPTION subscription_name;
```

## Parameters

| Parameter | Description |
| :-------- | :---------- |
| _subscription\_name_ | The name of the subscription to drop. |

## Example

```sql
-- Drop a subscription
DROP SUBSCRIPTION sub_users;
```
