---
title: "ALTER FUNCTION"
description: The `ALTER FUNCTION` command modifies the definition of a function. To use this command, you must own the function.
---

## Syntax

```sql
ALTER FUNCTION function( argument_type [, ...] )
    alter_option;
```

_`alter_option`_ depends on the operation you want to perform on the function. For all supported clauses, see the sections below.

## Clause

### `SET SCHEMA`

```sql
ALTER FUNCTION function( argument_type [, ...] )
    SET SCHEMA schema_name;
```

| Parameter or clause | Description                                                                                                                           |
| :------------------ | :------------------------------------------------------------------------------------------------------------------------------------ |
| **SET SCHEMA**      | This clause changes the schema of the function. To change a function's schema, you must also have CREATE privilege on the new schema. |
| _schema\_name_      | Specify the schema to which you want to change.                                                                                       |

```sql
-- Change the schema of the function named "test_func" to a schema named "test_schema"
ALTER FUNCTION test_func(INT) SET SCHEMA test_schema;
```
