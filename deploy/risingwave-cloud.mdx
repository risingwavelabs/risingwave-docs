## Prerequisites

* Create a [RisingWave Cloud account](https://cloud.risingwave.com/auth/signup/).
* If you plan to connect from a local SQL client, install [`psql`](/deploy/install-psql-without-postgresql) or another PostgreSQL-compatible client.

<Note>
For the full onboarding flow, see the [RisingWave Cloud quickstart](/cloud/quickstart).
</Note>
